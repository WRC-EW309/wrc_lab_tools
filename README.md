# WRC Lab Tools

Utility tools for WRC lab workspace.

## Installation

### Development Installation (Editable)

Install in editable mode so changes are immediately reflected:

```bash
pip install -e /home/wrc/.surfer
```

### Regular Installation

```bash
pip install /home/wrc/.surfer
```

## Usage

```python
import wrc_lab_tools

# Your code here
```

## Importing Surfer Library
1.) Ensure your virutal environment is loaded
2.) From your python script from surfer_lib.surfer import Surfer

## Development

To install with development dependencies:

```bash
pip install -e /home/wrc/.surfer[dev]
```
